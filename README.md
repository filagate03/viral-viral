# VIRAL OS

AI-инструмент для превращения одной идеи в поисковые запросы, анализ трендов YouTube, сюжетные сценарии и производственный гайд.

## Быстрый запуск локально
1. Дважды щёлкните `start-local.cmd` в корне проекта. Скрипт проверит наличие `npm`, установит зависимости (если нужно) и запустит `npm run dev`.
2. Откройте `http://localhost:5173` в браузере. Остановить сервер можно сочетанием `Ctrl + C` в появившемся терминале.

> Если хотите запустить всё вручную: `npm install` → `npm run dev`.

## Окружение
Создайте файл `.env` на основе `.env.example` и заполните ключи:
```
VITE_OPENAI_API_KEY=
VITE_OPENAI_PROJECT_ID=
VITE_YOUTUBE_API_KEY=
```
Секреты не коммитьте в репозиторий и не храните в открытом виде.

## Деплой на Netlify (RU)
1. **Подготовьте репозиторий.** Запушьте код на GitHub/GitLab/Bitbucket.
2. **Создайте сайт.** В Netlify нажмите *Add new site → Import an existing project* и выберите репозиторий.
3. **Настройте сборку.**
   - Build command: `npm run build`
   - Publish directory: `dist`
   - Node version: Netlify считает из `netlify.toml` (`NODE_VERSION = 22`), при необходимости оставьте значение.
4. **Добавьте переменные окружения.** В разделе *Site settings → Environment variables* укажите:
   - `VITE_OPENAI_API_KEY`
   - `VITE_OPENAI_PROJECT_ID`
   - `VITE_YOUTUBE_API_KEY`
5. **Запустите деплой.** Нажмите *Deploy site*. Netlify установит зависимости, соберёт проект и опубликует статику.
6. **Проверка.** После успешного билда откройте выдаваемый домен, убедитесь что UI загружается, формы работают (с корректными ключами API).

## Скрипты npm
- `npm run dev` — запуск Vite dev-сервера
- `npm run build` — проверка типов и прод- сборка
- `npm run preview` — локальный предпросмотр собранной версии
- `npm run lint` — запуск ESLint

## Структура проекта
```
src/
├── components/
│   ├── layout/      # Шаблон страницы и заголовок
│   ├── ui/          # Переиспользуемые элементы управления
│   └── features/    # Блоки приложения (форма, прогресс, аналитика)
├── context/         # Провайдер с глобальным состоянием
├── hooks/           # Кастомные хуки
├── services/        # API-клиенты и пайплайн
├── styles/          # Константы темы
├── types/           # Общие TypeScript интерфейсы
├── utils/           # Утилиты форматирования и прогресса
├── App.tsx          # Компоновка разделов
└── main.tsx         # Точка входа, провайдеры
```
